# Salary-Prediction
AI-based Salary Prediction System using Machine Learning. This project predicts employee salaries based on features like experience, education, job role, and demographics using regression models, with data preprocessing, visualization, and deployment-ready pipeline
# 💼 Salary Prediction System (AI/ML Project)

## 🚀 Overview

This project is a Machine Learning-based Salary Prediction System that estimates an individual's salary based on multiple features such as age, gender, education level, job title, and years of experience.

The goal is to solve a real-world problem of salary estimation using data-driven techniques and build an end-to-end ML pipeline.

---

## 🧠 Problem Statement

Organizations and individuals often need to estimate fair salary ranges. This project uses machine learning to predict salaries based on key attributes.

---

## 📊 Dataset

The dataset contains the following features:

* Age
* Gender
* Education Level
* Job Title
* Years of Experience
* Salary (Target Variable)

---

## ⚙️ Technologies Used

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* Matplotlib & Seaborn
* Pickle (Model Saving)

---

## 🔄 Project Workflow

1. Data Collection
2. Data Cleaning (handling missing values, removing outliers)
3. Feature Engineering (encoding categorical variables)
4. Data Visualization (EDA)
5. Model Training (Linear Regression, Random Forest)
6. Model Evaluation (MAE, R² Score)
7. Model Saving (Pickle)
8. Prediction System

---

## 📈 Model Performance

* Evaluated using:

  * Mean Absolute Error (MAE)
  * R² Score

(Random Forest performed better than Linear Regression)

---

## 📊 Visualizations

* Salary vs Experience Scatter Plot
* Salary Distribution
* Correlation Heatmap
* Boxplots for Outlier Detection

## 🧪 How to Run the Project

### 1. Clone Repository

```bash
git clone https://github.com/vijaymourya6464/salary-prediction.git
cd salary-prediction
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Train Model

```bash
python train.py
```

### 4. Run Prediction

```bash
python predict.py
```
